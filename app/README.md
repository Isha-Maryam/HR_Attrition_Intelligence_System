# HR Attrition Intelligence System

An advanced AI-powered decision support system designed to assist HR professionals in proactively identifying and managing employee turnover risks.

## Features
- **Employee Risk Estimation**: Predicts attrition probability using a fine-tuned XGBoost model.
- **Root Cause Extraction**: pinpoints specific drivers of turnover risk.
- **Retention Recommendation Engine**: Recommends customized, prescriptive intervention strategies.
- **Interactive UI**: A sleek, dark-themed dashboard built with Streamlit.

## Local Setup
1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   streamlit run app.py
   ```
